# 🐾 Zootierpflege-Verwaltungssystem

Willkommen! Dieses Projekt hilft uns, alle Informationen zur Zootierpflege übersichtlich zu organisieren.

## 📁 Struktur

- **daten/** - Alle Datendateien (Tierarten, Futtermittel, etc.)
- **webapp/** - Java Web-App zum Verwalten der Daten
- **docs/** - Dokumentation und Anleitungen

## 🚀 Schnelleinstieg

1. Schau in den `daten/` Ordner für bestehende Informationen
2. Nutze die Templates im `daten/templates/` Ordner für neue Einträge
3. Die Web-App wird später die Daten automatisch anzeigen
